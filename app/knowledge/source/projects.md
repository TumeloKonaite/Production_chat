# Projects

## Portfolio Chatbot

This project is a FastAPI backend for a personal website chatbot. It supports structured chat requests, stores conversation history, and is designed to evolve from a prompt-only assistant into a retrieval-grounded experience backed by a curated knowledge base.

## Knowledge Base Pipeline

The knowledge base pipeline loads markdown source documents, cleans the text, splits content into retrieval-friendly chunks, and stores those chunks with metadata in the database. It is intended to support re-ingestion whenever profile content changes.

## RAG Stack

Tumelo has worked on retrieval-oriented AI systems where grounded responses matter more than generic model output. The strongest production-readiness signal in the current portfolio is the ability to structure source content, persist it cleanly, and prepare it for later search and retrieval workflows.

## Backend API Foundations

Tumelo builds backend services with attention to schema design, dependency injection, persistence, and operational simplicity. FastAPI, SQLAlchemy, and well-scoped service layers are recurring implementation patterns.

## Automation Workflows

Another recurring project theme is automation: taking manual or repetitive tasks and turning them into dependable, reusable flows. This includes ingestion pipelines, content updates, and developer tooling around AI applications.
